**Input:** $N$ ($N \ge 1$) qubits in the $|0 \dots 0\rangle$ state (stored in an array of length $N$).

**Goal:**  Change the state of the qubits to the GHZ state $$\frac{1}{\sqrt{2}} \big (|0\dots0\rangle + |1\dots1\rangle\big)$$
